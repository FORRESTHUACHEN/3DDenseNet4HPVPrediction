# 3DDenseNet4HPVPrediction
Here is sourece codes, alongside the data for statistical analysis, and supporting materials for manuscript '**Non-invasive CT-based Deep Learning for Human Papillomavirus Status Prediction in Oropharyngeal Cancer**'

Pipeline of eatablished HPV prediction Network is shown as follow:
![Pipeline of established HPV prediction Network](https://github.com/FORRESTHUACHEN/3DDenseNet4HPVPrediction/blob/main/Figures/Figure1.png)

Study included three open access datasets are shown as follow:

Computed Tomography Images from Large Head and Neck Cohort [RADCURE](https://www.cancerimagingarchive.net/collection/radcure/)

Head-Neck-PET-CT[H&N PET/CT](https://www.cancerimagingarchive.net/collection/head-neck-pet-ct/)

HEAD-NECK-RADIOMICS-HN1 [HN1](https://www.cancerimagingarchive.net/collection/head-neck-radiomics-hn1/)
